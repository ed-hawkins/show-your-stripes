# show-your-stripes
Graphics for the Show Your Stripes initiative - see www.showyourstripes.info

File names indicate the REGION - COUNTRY - SUB_COUNTRY - TIME_PERIOD - DATA_SOURCE - with/without labels

For DATA_SOURCE:
BK = Berkeley Earth
MO = UK Met Office
NO = NOAA
DW = DWD
MF = Meteo-France
ZA = ZAMG
BO = Bolin Centre
MS = MeteoSwiss
